XooMLAssociation Class

Defined in: scripts/XooMLAssociation.js:1

XooMLAssociation is a minimal interface to represent a XooML2 association within a fragment.

Throws NullArgumentException when GUID, name, or isUpgradeable is null.
Throws InvalidTypeException if GUID, or name is not a String and if isUpgradeable is not a boolean.
Throws InvalidGUIDException if GUID is not a valid GUID.

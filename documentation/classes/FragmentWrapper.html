FragmentWrapper Class

Defined in: scripts/FragmentDriver.js:1

Constructs a FragmentWrapper for a XooML fragment. In the following cases.

Case 1: xooMLFragmentString is given and is used as the XooMLFragment.
Case 2: associations, xooMLUtilityURI, itemUtilityURI, syncUtilityURI, groupingItemURI are given and used to create a new XooMLFragment with the given data.

Throws NullArgumentException when options is null.
Throws MissingParameterException when options is not null and does not have the necessary arguments for any given case.

createAssociation

(
  • options
)
String

Defined in scripts/FragmentDriver.js:771

Creates an association based on the given options and the following cases.

Case 1: Simple text association declared phantom.
Case 2: Link to existing non-grouping item, phantom.
Case 3: Link to existing non-grouping item, real.
Case 4: Link to existing grouping item, phantom.
Case 5: Link to existing grouping item, real.
Case 6: Create new local non-grouping item.
Case 7: Create new local grouping item.

Throws NullArgumentException when options, callback is null.
Throws InvalidTypeException when options is not an object and callback is not a function.
Throws MissingParameterException when an argument is missing for an expected case.

Parameters:

  • options Object

    Data to create an new association for.

    • displayText. String

      Display text for the association. Required in all cases.

    • itemOrXooMLFragmentURI String

      URI of the item or XooMLFragment depending on the case. Required for cases 2, 3, 4 & 5.

    • localItemRequested Boolean

      True if the local item is requested, else false. Required for cases 2, 3, 4, & 5.

    • xooMLUtilityURI String

      URI of the XooML utility for the association. Required for cases 4 & 5.

    • itemName String

      Name of the new local non-grouping/grouping item. Required for cases 6 & 7.

    • itemType String

      Type of the new local non-grouping/grouping item. Required for case 6.

Returns:

String:

GUID of the newly created association.
